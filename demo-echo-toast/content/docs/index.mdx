## What is Echo Toast?

Echo Toast was created to be an alternative to multiple existing libraries like React Toastify, React Hot Toast, Sonner, etc. It was created with native CSS animations and Lucide React Icons.

## State Management Architecture

It was created with the publisher/subscriber pattern, know as [Observer Pattern](https://refactoring.guru/design-patterns/observer), allowing user add a ```<Toaster/>``` component that handle all the information of the toasts. Also its an imperative API allowing you call all the functions to render a new toast from anywhere in your code, if ```<Toaster/>``` is added, it will render it.
